# True EduCare - Website

This is a React + Vite + TailwindCSS starter website for **True EduCare**.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Web3Forms Contact Form

The contact form is set to use Web3Forms with a **dummy access key**. Replace `DUMMY_ACCESS_KEY_REPLACE` in `src/pages/Contact.jsx` with your real Web3Forms access key.

Web3Forms docs: https://web3forms.com

## Deploy

Recommended: Vercel or Netlify. Connect your GitHub repo and deploy. Don't forget to add a custom domain.

## Notes

- Update contact details and email in `src/components/Footer.jsx`.
- Add logos and images in `src/assets`.
